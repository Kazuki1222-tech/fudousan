# ふどうさん（不動産かんたん計算ツール）

坪⇄㎡の変換、坪/㎡単価の算出、住所からの土地価格推定（TBD）、学区検索（町名・学校）をまとめた、モバイルフレンドリーなWebツールです。  
UI は **shadcn/ui** をベースに構築し、**GitHub Pages** に静的公開、**GitHub Actions** で自動デプロイします。

## 主な機能

### 1) 坪・㎡ 変換
- 坪 → ㎡、㎡ → 坪 を即時変換
- 端数処理（小数点桁数）を見やすく表示

### 2) 単価計算（坪単価・㎡単価）
- 土地の価格と面積（坪 または ㎡）を入力すると、**坪単価** と **㎡単価** を同時に算出
- 単位切替（坪/㎡）に対応し、比較検討をスムーズに

### 3) 住所から推定価格（TBD）
- 住所を入力すると、その地点の**推定価格**を表示（仕様・ロジック検討中）
- 推定は参考値です（詳細は「免責事項」参照）

### 4) 学区検索（町名・学校）
- 自治体が公開している**学区PDF**を、検索しやすい**テーブル（データ）**に整理して表示
- 町名を選択すると、該当する**小学校・中学校**を表示（町名 → 学校）
- 学校を選択すると、対象となる**町名一覧**を表示（学校 → 町名）
- まずは**町名単位**に対応（将来的に丁目・番地などの精度向上を検討）

## 使い方

### 坪・㎡変換
1. 変換したい数値を入力
2. 坪/㎡ を切り替える（必要に応じて）
3. 変換結果を確認

### 単価計算（坪単価・㎡単価）
1. 土地価格を入力
2. 面積を入力（坪 もしくは ㎡）
3. 坪単価・㎡単価を確認

### 住所から推定価格（TBD）
仕様・ロジックが固まり次第、使い方を追記します。

### 学区検索（町名・学校）
1. 自治体（対象エリア）を選択（対応している場合）
2. 町名から探す: 町名を選択 → 小学校・中学校を確認
3. 学校から探す: 学校を選択 → 対象の町名一覧を確認

## 計算仕様

### 坪⇄㎡
- **1 坪 = 3.305785 ㎡**
- ㎡ → 坪: `㎡ ÷ 3.305785`
- 坪 → ㎡: `坪 × 3.305785`

### 単価
- 坪単価: `価格 ÷ 面積(坪)`
- ㎡単価: `価格 ÷ 面積(㎡)`

## 推定ロジック（TBD）
住所からの推定価格は、データソースや算出方法（例: ジオコーディング、参照する地価/取引データ、補正方法など）を検討中です。決まり次第このセクションを更新します。

## 学区データ（概要）
- 学区情報は、各自治体が公開しているPDF等の資料を元に、アプリ内で参照できる形（例: JSON/CSV）へ整理して利用します
- 学区の境界・指定校は変更される可能性があります。最新情報は自治体の公式資料もあわせて確認してください

## ローカル開発
プロジェクトの構成によりコマンドは異なります。`package.json` の `scripts` に合わせて実行してください。

例（npm の場合）:
```bash
npm install
npm run dev
```

例（pnpm の場合）:
```bash
pnpm install
pnpm dev
```

## デプロイ（GitHub Pages + GitHub Actions）
- `main`（またはデフォルトブランチ）への push をトリガーに、GitHub Actions が build → GitHub Pages へ deploy します（想定）
- 初回は GitHub の `Settings → Pages` で `Source: GitHub Actions` を選択してください

## プライバシー
- 静的サイトとして動作する想定のため、入力した住所・金額・面積等をサーバへ保存しません
- ただし、住所推定機能で外部APIを利用する場合、入力した住所が当該サービスへ送信される可能性があります（TBD）

## 免責事項
- 本ツールの結果（単価・推定価格など）は**参考値**であり、鑑定評価・査定・投資判断を代替するものではありません
- 正確性・完全性・特定目的適合性を保証しません。最終判断はご自身で行ってください
- 学区情報は自治体資料を元に整理していますが、変更・誤差があり得ます。必ず自治体の公式資料で最終確認してください
